
1.- Verify Status:
ps aux | grep mysqld | grep -v grep

2.- Start Service:
sudo service mysql start

3.- Open Client:
sudo mycli

4.- Test database:
sudo mysql -u root -p classicmodels < DarioGarza_Homework4.sql | tee output.txt

5.- Upload Database:
sudo mysql miskatonic_university < miskatonic_ddl.sql

6.- Install Jupyter with Python connector:
pip install jupyter notebook mysql-connector-python pandas matplotlib seaborn

7.- Launch Jupyer in browser:
jupyter notebook --ip=0.0.0.0 --port=8888 --no-browser
jupyter server list

8.- Export schema
sudo mysqldump --no-data miskatonic_university > schema.sql
